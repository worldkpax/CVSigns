# CVSigns

Desktop-приложение на Python для обнаружения и классификации российских дорожных знаков в реальном времени по видеопотоку с локальной моделью `Ultralytics YOLO`.

## Возможности

- веб-камера в реальном времени
- открытие видеофайлов
- открытие одиночных изображений
- отрисовка `bounding box`, класса и confidence
- список последних обнаруженных знаков
- счетчик объектов на текущем кадре
- отображение FPS
- настройка `confidence` и `IoU`
- выбор устройства `auto` / `cpu` / `cuda`
- сохранение текущего кадра
- запись обработанного видео
- корректная обработка отсутствующей модели без падения приложения

## 1. Создание виртуального окружения

```bash
python3.11 -m venv .venv
source .venv/bin/activate
```

Для Windows:

```powershell
py -3.11 -m venv .venv
.venv\Scripts\activate
```

## 2. Установка зависимостей

```bash
pip install --upgrade pip
pip install -r requirements.txt
```

## 3. Куда положить модель

По умолчанию приложение ищет веса по пути:

```text
models/russian_traffic_signs.pt
```

Положите туда локальный файл модели `Ultralytics YOLO .pt`, обученный на российских дорожных знаках. Если путь другой, его можно выбрать в интерфейсе через кнопку `Выбрать модель`.

## 4. Запуск приложения

```bash
python main.py
```

При первом запуске:

1. Поместите модель в `models/russian_traffic_signs.pt` или выберите `.pt` вручную.
2. Откройте изображение, видео или подключите камеру.
3. Нажмите `Старт`.

Файл `config.json` будет создан автоматически при закрытии приложения и сохранит последний путь к модели и параметры интерфейса.

## 5. Сборка exe через PyInstaller

```bash
pyinstaller CVSigns.spec
```

Готовая сборка появится в директории `dist/CVSigns/`.

Если модель не встроена в сборку, просто положите файл весов рядом со структурой проекта в папку `models/`.

## Примечания

- Имена классов берутся из `model.names`.
- Если `model.names` отсутствует или поврежден, приложение безопасно использует fallback вида `class_0`, `class_1` и так далее.
- Для режима `cuda` требуется доступный CUDA-совместимый `torch`.
